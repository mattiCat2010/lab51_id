# Project Structure Guide

This document details the layout of the `lab51_id` project, explaining where key components reside.

## Codebase

The root directory contains configuration files for Docker, CI/CD, and project-wide settings.

## Backend

The backend code, written in Node.js/Express, handles API logic, database interaction, and business rules.

## Frontend

The frontend code, built using React and compiled via Vite, provides the user interface.
